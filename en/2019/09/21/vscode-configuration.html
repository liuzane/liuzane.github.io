VSCode Editor Settings:

  • Indentation Indentation
    detectIndentation - Controls whether to automatically detect “editor.tabSize” and “editor.insertSpaces” based on file content when opening files.
    tabSize - Number of spaces a tab is equal to. When “editor.detectIndentation” is on, this setting will be overridden based on file content.
    insertSpaces - Insert spaces when pressing “Tab”. When “editor.detectIndentation” is on, this setting will be overridden based on file content.
  • Line Endings
    Default line ending character.
    ValueDescription
    \nLF (Linux)
    \r\nCRLF (Windows)
    autoUses operating system specific end of line character.
  • Auto Save Auto Save
    ValueDescription
    offTo disable auto save.
    afterDelayTo save files after a configured delay (default 1000 ms).
    onFocusChangeTo save files when focus moves out of the editor of the dirty file.
    onWindowChangeTo save files when the focus moves out of the VS Code window.
  • Auto Formatting Formatting
    formatOnSave - Format code files when saving
    formatOnType - Format the line of code after typing
    formatOnPaste - Format pasted content
  • Default Formatter Default formatter
    dbaeumer.vscode-eslint - ESLint Plugin.
    esbenp.prettier-vscode - Prettier - Code formatter.
  • Word Wrap
    Whether to wrap lines when code length exceeds editor visible width
  • Expand Abbreviations Emmet
    Use shortcut syntax to expand tags, e.g., ul>li*3>span.hello
  • Sticky Scroll Sticky Scroll
    https://code.visualstudio.com/updates/v1_85#_sticky-scroll-in-trees
  • Others
    dragAndDrop - Control whether the editor allows drag-and-drop to move the selected content.
    ignoreRecommendations - Set to true to disable extension recommendation push, see details here
    proxy - Set proxy for editor
