VSCode编辑器设置：

  • 缩进 Indentation

    detectIndentation - 控制打开文件时是否根据文件内容自动检测“editor.tabSize”和“editor.insertSpaces”。
    tabSize - 制表符等于空格数。 当“editor.detectIndentation”打开时，此设置将根据文件内容被覆盖。
    insertSpaces - 按“Tab”时插入空格。 当“editor.detectIndentation”打开时，此设置将根据文件内容被覆盖。

  • 行结束符

    默认的行结束符。

    值说明
    \nLF (Linux)
    \r\nCRLF (Windows)
    autoUses operating system specific end of line character.
  • 自动保存 Auto Save
    值说明
    offto disable auto save.
    afterDelayto save files after a configured delay (default 1000 ms).
    onFocusChangeto save files when focus moves out of the editor of the dirty file.
    onWindowChangeto save files when the focus moves out of the VS Code window.
  • 自动格式化 Formatting

    formatOnSave - 保存时格式化代码文件
    formatOnType - 输入后格式化该行代码
    formatOnPaste - 格式化粘贴内容

  • 默认格式化插件 Default formatter

    dbaeumer.vscode-eslint - Eslint Plugin.
    esbenp.prettier-vscode - Prettier - Code formatter.

  • 换行

    代码长度超过编辑器可视宽度是否换行

  • 展开缩写 Emmet

    使用快捷语法展开标签，例如 ul>li*3>span.hello

  • 粘滞滚动 Sticky Scroll

    https://code.visualstudio.com/updates/v1_85#_sticky-scroll-in-trees

  • 其他

    ignoreRecommendations - 设置true可以禁止推荐插件推送， 详情见这里
    proxy - 编辑器设置代理
