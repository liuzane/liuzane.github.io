JavaScript Array 对象

语法

方法集合：

这里收集部分常用但又不是太清楚详细用法的方法

  • reduce

    参数描述
    total必需。初始值, 或者计算结束后的返回值。
    currentValue必需。当前元素。
    currentIndex可选。当前元素的索引。
    arr可选。当前元素所属的数组对象。
    initialValue可选。传递给函数的初始值。

    整合 map 和 filter 功能的例子

  • slice

    参数描述
    start必需。开始元素的索引。
    end可选。结束元素的索引。

    方法创建新数组。它不会从源数组中删除任何元素。
    如果结束参数被省略，slice() 会切出数组的剩余部分。
